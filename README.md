# TIF Normalizer

TIF formatındaki grafik/kayıt görüntülerindeki çizgilerin renk kodlarını normalize eden web uygulaması.

## Özellikler
- TIF dosyası yükleme
- Otomatik çizgi renk tespiti
- Rolling window ile renk normalizasyonu
- Orijinal vs. normalleştirilmiş görsel karşılaştırma
- İşlenmiş görüntüyü indirme
